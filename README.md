# text-editor-on-dos

16年C语言课程设计，纯c语言实现的dos下图形化文本编辑器，界面全部使用图形驱动API绘制完成



### 编译运行

1. 使用Windows XP或者安装DOSBox模拟环境
2. 安装Turboc 2.0
3. 使用Turboc 2.0加载editor.prj，选择编译模式为 huge
4. 编译运行
